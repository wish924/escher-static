埃舍尔团队由互联网行业顶尖人物组建，拥有丰富的从业经历、行业资源和众多高科技人才储备，公司总部设在昆明，

影视制作部

VR内容编导(昆明) 1人

岗位职责

1）负责完成公司所需要的行业信息及专题活动视频拍摄及后期制作；

2）负责视频制作时所需的音频、影像等素材的制作、搜集及整理；

3）负责把控采编的视频、影像质量；

4）对设备进行日常的保养、维护，保证设备的正常使用；

5）负责独立完成相关视频内容的策划、撰稿、录制工作，并对指定素材进行摄像。

任职要求

1）熟悉行业信息和专题活动的摄影、摄像及相关技能，能处理摄影、摄像工作中的技术问题，并对光效、布景等相关方面有自己的创意与想法；

2）有独立拍摄和剪辑完整视频的能力；

3）熟练使用后期制作过程中的相关软件，如若会使用主流视频包装剪辑软件更佳（MAYA或者3DMAX、After Effect、Flash、Premiere等）；

4）有责任心，有团队协作意识和创新精神。愿意长期从事该行业，能长期出差，并能承受一定的工作压力；

5）熟悉宣传片、专题片、片头、广告片等类型特效制作者优先考虑。
